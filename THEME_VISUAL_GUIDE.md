## Typography

### Headers
- **Font**: Century Gothic
- **Weight**: Bold
- **Sizes**: 
  - Form titles: 24-30pt
  - Hero titles: 32pt
  - Dashboard KPI: 28pt
- **Color**: Gold (RGB 218, 165, 32)

### Labels & Captions
- **Font**: Segoe UI
- **Weight**: Regular or Bold (10pt)
- **Color**: Light Gray (RGB 200, 200, 200)

### Body Text
- **Font**: Segoe UI
- **Weight**: Regular
- **Size**: 9-11pt
- **Color**: White or Light Gray

---

## Component Styling

### Input Fields
- **Height**: 40-42px
- **Border Radius**: 6-8px
- **Fill Color**: RGB(45, 45, 45)
- **Border Color**: RGB(50, 50, 50)
- **Focused Border**: Gold (RGB 218, 165, 32)
- **Focused Fill**: RGB(40, 40, 40)
- **Text Color**: White

### Buttons
- **Border Radius**: 8-10px
- **Fill Color**: Gold (RGB 218, 165, 32)
- **Text Color**: Black
- **Hover Fill**: Bright Gold (RGB 255, 215, 0)
- **Font**: Segoe UI, Bold, 11pt
- **Height**: 45-60px

### Links
- **Color**: Gold (RGB 218, 165, 32)
- **Font**: Segoe UI, 10-11pt
- **Visited**: Gold (same as normal)

### Panels & Containers
- **Border Radius**: 20-40px
- **Fill Colors**:
  - Dark: RGB(20, 20, 20)
  - Secondary: RGB(35, 35, 35)
- **Shadow Color**: Gold (RGB 218, 165, 32)
- **Shadow Depth**: 20-35px

## Responsive Design Notes

### Desktop (1920x1080+)
- ? Full layout with side-by-side panels
- ? Centered card positioning
- ? Proper spacing on all elements

### Tablet (1280x720+)
- Forms may need horizontal scrolling
- Panel widths scale proportionally
- Input fields maintain 380-420px width

### Mobile
- Not optimized (designed for desktop/web)
- Full-screen required for proper display

---

## Accessibility Features

1. **Color Contrast**: Gold on dark provides WCAG AA compliance
2. **Focus States**: Clear gold border on input focus
3. **Text Size**: Minimum 11pt for body text
4. **Button Size**: Minimum 45px height for touch targets
5. **Labels**: Clearly associated with form fields

---

## Browser Compatibility

- **Framework**: WinForms (.NET Framework 4.7.2)
- **Controls**: Guna.UI2.WinForms
- **Colors**: Standard GDI+ Color.FromArgb()
- **Fonts**: System fonts (Century Gothic, Segoe UI)

---

## Animation & Transitions

### Current State
- Button hover: FillColor change to brighter gold
- Focus state: Border color change to gold
- No animations (smooth transitions planned for v2)

### Recommended Future Additions
- Smooth button hover transition (300ms)
- Fade-in animation on form load
- Button press animation (scale effect)
- Input field focus animation (border glow)
